--- layout: layout.njk title: Baseline Support for Intercom --- Are you using Intercom to communicate with your customers? With the Baseline for Intercom integration you can easily review who has access to your Intercom workspace. ## Setting up Intercom for Baseline Setting up Intercom for Baseline is super simple. Once you have kicked off a new Baseline audit, you can simply click on the Intercom card on the services selection page. You need to be workspace admin if you want to be able to baseline your Intercom workspace. ## Required OAuth permissions Baseline relies on the following permissions to audit your Intercom account: | OAuth Scope | |--------------------------| | Read admins | | Update admins | | Read one admin | | Read admin activity logs |
